Garden Status

The canonical status record for the Acacia Garden Codex is maintained in STATUS.json at the repository root.

Schema definition:
STATUS.schema.json

Any derived dashboards, pages or visualisations are secondary to this JSON source.

Typical Fields (Conceptual)

This is a human-readable explanation of common fields; the exact values live in STATUS.json:

  • version — current status document version.
  • updated_at — last update timestamp (ISO-8601).
  • anchor — the Garden anchor string (e.g., HKX277206).
  • cycles — summary of known cycles and their health.
  • signals — high-level flags (ok, degraded, experimental, etc.).
  • notes — free-form human notes attached to the current state.

For machines: validate STATUS.json with STATUS.schema.json.
For humans: treat this page as a guide, not as the source of truth.
