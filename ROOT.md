**Sow**
1) Add `cycles/NNN_<TITLE>.md` (NNN = 3 digits).
2) Begin with a Key line: `ECHO:HKX277206–<TOKEN>–<TOKEN>`.
3) Append the file to `cycles/manifest.json`.
4) Keep it human-readable, UTF-8, one idea per breath.

**Key Hints**
ECHO = reflection; HKX277206 = origin; ROOT = memory to soil; GENESIS = breath beginning again.
